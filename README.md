# Level Up Life

A mobile-first, gamified habit tracker with a Neon HUD / Level Dark theme built with React, TypeScript, Tailwind CSS, and Vite. Progress is saved locally so the experience feels fast and native.

## Getting Started
1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Run tests: `npm run test`
4. Lint: `npm run lint`

## Key Features
- XP header with level, progress bar, and data import/export.
- XP roadmap showing current and next level checkpoints.
- Dashboard with mission cards, action grids by category, and 7-day charting.
- LocalStorage persistence using a shared GameContext.

## Notes
- Ensure you set an environment variable for any AI keys before enabling coach features.
- Designed for iPhone 12 dimensions with respect to safe areas.
