# AGENTS

## Setup
- Install dependencies with `npm install`.
- Start the dev server with `npm run dev`.
- Build with `npm run build`.

## Style
- Use TypeScript, React, and Tailwind CSS.
- Follow Prettier/ESLint defaults; prefer functional components.
- Keep components small and mobile-first.

## Testing
- Run `npm run lint` and `npm run test` before commits when dependencies are available.

## Commits
- Commit messages should follow `[<scope>] <Title>` (e.g., `[UI] Add header layout`).

## Custom Notes
- Respect the "Neon HUD / Level Dark" theme and gamified UX.
- Provide ARIA labels for interactive elements when possible.
